# How to test
For send data to database:
```
curl -X POST -H "Content-Type: application/json" -d '{"key": "key1", "value":"value1"}' localhost:8080/set
```
For getting data:
```
curl -H "Content-Type: application/json" -d '{"key": "key1"}' localhost:8080/get
```
